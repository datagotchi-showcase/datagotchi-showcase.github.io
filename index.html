JSON Input

Define layers as an ordered array. Each layer uses a URL, a zIndex for ordering, and meta properties for position and size. Alpha blending is supported via meta.alpha.

How to Use

        
    
          
  1. Paste or edit the JSON in the editor. Each object is a sprite layer.
    2. 
          
  2. Click Compose to load images asynchronously and render the stack by zIndex.
    3. 
          
  3. Use Download PNG to save the final composite or Copy Base64 for APIs.

Layer Schema

          
[
  {
    "image": "https://example.com/layer.png",
    "zIndex": 0,
    "meta": {
      "xOffset": 0,
      "yOffset": 0,
      "width": 100,
      "height": 100,
      "alpha": 1
    }
  }
]
